### M1 不可妥协的验收条件

- `WorkerExited(pid)` 只能观察退出，不能调用 `waitpid`、`Wait4`、`exec.Cmd.Wait` 或其它 reap primitive。
- Darwin 优先使用 `kqueue + EVFILT_PROC + NOTE_EXIT`。
- **禁止**把 `Wait4(..., WNOHANG)`描述为 non-reaping：子进程退出后它仍会消费 exit status。
- `ReapWorker(pid)` 必须是唯一实际 `Wait4` 位置，并在 group cleanup 之后执行。
- 回归测试必须证明：收到 exit notification 后，PID 在 `ReapWorker` 前仍保持存在/可 reap；然后 `ReapWorker` 成功取得真实 exit code。
- M1 mutation gate 必须拒绝真实 eager reap，不只是 `ReapWorker` 与 `GroupMembers` 方法名的 proxy call order。
